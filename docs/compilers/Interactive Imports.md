# 脚本/交互中的导入

## 指导原则

由 ```#load``` 包含的文件各自位于单独的词法作用域中。
声明（包括别名）优先于 using（包括 using static），因为它们表达了意图。

## 过程

步骤（从上到下） | 绑定 Using | 绑定代码 | #load
----------------------|---------------|--------------|------
在当前提交/文件中查找声明（包括别名） | 是，排除别名 | 是，在提交中 | 是，在文件中
在前面的提交中查找声明（包括别名），按顺序 | 是 | 是 | 否
在宿主对象（如果有）中查找成员 | 是 | 是 | 是
在全局命名空间中查找声明 | 是 | 是 | 是
在当前提交/文件和所有前面提交的 using（除别名外）中查找（一次全部，不循环） | 否 | 是，在所有提交中 | 是，仅在文件中
在当前提交和所有前面提交的全局导入中查找（一次全部，不循环） | 否 | 是 | 是
