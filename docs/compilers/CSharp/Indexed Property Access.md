# 索引属性访问

[此为占位内容。我们需要在此处补充更多文档]

在 VB 中声明的索引属性可以在 C# 中使用。我们应该详细描述我们的做法。

实际上，似乎除非在 VB 中加上 [ComVisible] 属性，否则无法使用索引属性语法。

如果一个类型具有 `[System.Runtime.InteropServices.ComImportAttribute]` 属性，并且具有一个或多个参数化属性，则可以从 C# 引用这些属性（前提是它们具有适当的可见性等）。此功能的主要场景是支持通常包含此类属性的 COM 库。在此功能出现之前，只能通过其方法名（P_set、P_get）调用此类属性的访问器。但也可以在 VB 中声明此类属性，我们通常用于测试此功能（注意，当指定 `[ComImport]` 属性时，还应使用有效的 `GUID` 作为参数指定 `[System.Runtime.InteropServices.GuidAttribute("…")]` 属性，否则会出现编译时错误）。

注意，可以声明多个具有不同名称的参数化属性，也可以为每个名称声明多个重载签名。在后一种情况下，具有相同名称的属性形成属性组（类似于 C# 规范中解释的方法组概念）。为了确定要调用的特定属性，C# 使用重载解析，其规则与索引器调用中的重载解析完全相同。
