布尔值的表示
================================

C# 和 VB 编译器分别用单字节值 `1` 和 `0` 表示 `true`（`True`）和 `false`（`False`）的 `bool`（`Boolean`）值，并假定它们正在处理的任何布尔值都限于由这两个基础值表示。ECMA 335 CLI 规范允许用任何非零值表示"真"布尔值。如果您使用基础表示不是 `0` 或 `1` 的布尔值，可能会得到意外的结果。这可能发生在 C# 的 `unsafe` 代码中，或通过与允许其他值的语言进行互操作时。为避免这些意外结果，程序员有责任规范化此类传入的值。
