### 笔记

IVariableDeclarationOperation：
 - 脚本模式下，IOperation 中对本地声明使用 `using` 时应如何处理？

VariableDeclarationKind
 - `Default` 是合适的种类吗？是否应改用 `Other`？
 - `Default` 的文档说明不够清晰，因为所有声明本身也是声明。
 - 是否应为 foreach、模式、using 语句添加其他种类？
 - 决定：添加新的父节点以表示 `IUsingDeclarationOperation`，用于作为 `IVariableDeclarationGroupOperation` 的父节点，同时移除 `VariableDeclarationKind`。

Simplifier.AddImportsAnnotation
 - 应该使用 `Using` 还是 `Imports`？
 - 在存在冲突时，我们已有使用 C# 术语的先例，但现有的 using 相关内容也使用了 Imports。我们将保留该名称。

我们开会的时间是否太晚了？
 - 可能是。未来我们应该考虑在询问模式结束之前召开这些会议，而不是每月一次。
