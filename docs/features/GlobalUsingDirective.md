全局 using 指令
=========================

*全局 using 指令*功能通过可选的 `global` 关键字扩展了 using 指令语法，该关键字可位于 `using` 关键字之前。
全局 using 指令的作用域跨越程序中的所有编译单元。

提案：https://github.com/dotnet/csharplang/blob/master/proposals/GlobalUsingDirective.md

功能分支：https://github.com/dotnet/roslyn/tree/features/GlobalUsingDirective

测试计划：https://github.com/dotnet/roslyn/issues/51307
