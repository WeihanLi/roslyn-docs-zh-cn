表达式变量
=========================

*表达式变量*功能扩展了 C# 7 中引入的功能，允许在字段初始化器、属性初始化器、构造函数初始化器和查询子句中使用包含表达式变量（out 变量声明和声明模式）的表达式。

有关更多信息，请参见 https://github.com/dotnet/csharplang/issues/32 和
https://github.com/dotnet/csharplang/blob/main/proposals/csharp-7.3/expression-variables-in-initializers.md。

功能当前状态：

[X] 允许在字段初始化器中使用
[X] 允许在属性初始化器中使用
[ ] 允许在构造函数初始化器中使用
[X] 允许在查询子句中使用
